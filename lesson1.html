Lesson 1: Introduction to HTML

In this lesson, you will learn the basics of HTML (Hypertext Markup Language), which is the standard language for creating web pages.

What is HTML?

HTML is the language used to structure content on the web. It defines the elements and structure of a web page using tags.

HTML Structure

<html>
    <head></head>
    <body>
        <h1>Welcome to My Website</h1>
        <p>This is my first webpage!</p>
    </body>
</html>

Practice:

Create a basic HTML page with a heading and a paragraph. Save it as index.html and submit it after you're done!
