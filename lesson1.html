Lesson Overview

In this lesson, we will cover how to set up Visual Studio Code Insiders Edition as your code editor. This tool will help you manage your code and enhance your coding experience.

Step 1: Installing Visual Studio Code Insiders Edition

Follow these steps to install Visual Studio Code Insiders Edition on your computer:

  1. Visit the Visual Studio Code Insiders website.
  2. Download the installer for your operating system (Windows, macOS, or Linux).
  3. Run the installer and follow the on-screen instructions to complete the installation.

Once installed, open VS Code Insiders and set it up by installing useful extensions for a better coding experience.

Step 2: Installing Essential Extensions

Install the following extensions to enhance your coding experience:

  1. Open Visual Studio Code Insiders.
  2. Go to the Extensions view by clicking the Extensions icon in the Activity Bar on the side of the window or by pressing Ctrl+Shift+X.
  3. Search for and install the following extensions:
    • Live Server by Ritwick Dey
    • GitHub Copilot by GitHub
    • GitHub Copilot Chat by GitHub
