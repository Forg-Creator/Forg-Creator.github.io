Lesson Overview

In this lesson, we will cover how to set up Visual Studio Code (VS Code) as your code editor and GitHub for version control. These tools will help you manage your code and collaborate with others.

Step 1: Installing Visual Studio Code

Follow these steps to install Visual Studio Code on your computer:

  1. Visit the Visual Studio Code website.
  2. Download the installer for your operating system (Windows, macOS, or Linux).
  3. Run the installer and follow the on-screen instructions to complete the installation.

Once installed, open VS Code and set it up by installing useful extensions (such as the Live Server extension) for a better coding experience.

Step 2: Installing Git and Setting Up GitHub

GitHub is a platform for version control and collaboration. Before you can use GitHub, you need to install Git on your computer.

  1. Visit the Git website and download the installer for your operating system.
  2. Run the installer and follow the setup instructions.
  3. Once Git is installed, create a free GitHub account at GitHub.
  4. After signing up, generate an SSH key to link your GitHub account with your local machine.

Check out GitHub’s guide to SSH keys for more help.

Step 3: First Git Commit

Now that you have both Visual Studio Code and GitHub set up, let’s make your first commit:

  1. Open VS Code, and create a new folder for your project.
  2. Open the integrated terminal in VS Code (Ctrl+` or Cmd+` on macOS).
  3. In the terminal, run the following commands to initialize your repository:
    4. git init
  4. Next, create a file named index.html and add some HTML content.
  5. Now, stage and commit your changes:
    6. git add .
    git commit -m "Initial commit"

Congratulations! You've just created your first Git repository and committed your changes.
