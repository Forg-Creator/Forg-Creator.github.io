Lesson Overview

In this lesson, we will learn the basics of HTML, the language used to structure and display content on the web. HTML stands for HyperText Markup Language, and it’s used to create webpages by structuring text, images, links, and other elements.

What is HTML?

HTML is a markup language, which means it’s used to describe the structure of content. It’s made up of various elements such as tags and attributes. Let’s look at an example:

<!DOCTYPE html>
<html>
    <head>
        <title>My First Webpage</title>
    </head>
    <body>
        <h1>Welcome to My Webpage</h1>
        <p>This is a paragraph of text.</p>
    </body>
</html>

In this example:

  • <html> is the root element of the HTML document.
  • <head> contains metadata, like the title of the page.
  • <body> contains the visible content of the webpage.
  • <h1> creates a top-level heading.
  • <p> is used to define a paragraph.

Basic HTML Tags

Let’s look at some of the basic HTML tags that will be used throughout the course:

  • <h1> to <h6>: These are heading tags, with <h1> being the largest and most important, and <h6> being the smallest.
  • <p>: Defines a paragraph of text.
  • <a>: Defines a hyperlink to another webpage or resource.
  • <img>: Embeds an image.
  • <ul>, <ol>, <li>: Used for creating unordered and ordered lists.
  • <div>: A container element used to group content.

Creating Your First HTML Page

Now, let’s create a simple HTML page together. Open your Visual Studio Code editor and create a new file named index.html. Add the following code:

<!DOCTYPE html>
<html>
    <head>
        <title>My First Webpage</title>
    </head>
    <body>
        <h1>Welcome to My First Webpage</h1>
        <p>Hello, world! This is my first HTML page.</p>
    </body>
</html>

Save your file and open it in your web browser to see your first webpage!
