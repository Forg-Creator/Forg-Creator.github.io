Lesson 2: HTML Tags & Elements

In this lesson, you'll learn about the most important HTML tags and elements used in web development.

Common HTML Tags

  • <h1> to <h6> - Headings
  • <p> - Paragraph
  • <a> - Link
  • <ul>, <ol>, <li> - Lists
  • <img> - Image

Practice:

Create a webpage with a heading, a paragraph, and a list. Include a link and an image. Here's a simple example:

    <html>
        <body>
            <h1>My Favorite Websites</h1>
            <p>Here are some websites I love to visit.</p>
            <ul>
                <li><a href="https://www.example.com">Example Website</a></li>
                <li><a href="https://www.google.com">Google</a></li>
            </ul>
            <img src="image.jpg" alt="A beautiful picture">
        </body>
    </html>

Once you're done, move on to Lesson 3!
