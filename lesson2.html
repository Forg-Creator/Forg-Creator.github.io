Lesson Overview

Previously, you set up your coding environment. Now, you'll learn how to structure your web pages using HTML elements.

This lesson focuses on organizing your webpage with HTML elements like sections, lists, and links. These are the building blocks for all web pages.

HTML Sections

HTML allows you to break up your webpage into sections using <section> and <div> tags. This helps organize your content.

<section>
    <h2>Introduction</h2>
    <p>Welcome to my webpage!</p>
</section>

Lists

Use <ul> for unordered lists, <ol> for ordered lists, and <li> for list items.

<ul>
    <li>HTML Basics</li>
    <li>CSS Styling</li>
</ul>

Links

Links are created using the <a> tag. You can link to other pages or external websites.

<a href="https://www.example.com">Visit Example</a>

Challenge

Structure a page with the following:

  1. A heading with an introduction paragraph.
  2. A list of your favorite hobbies.
  3. A link to a website you like.
