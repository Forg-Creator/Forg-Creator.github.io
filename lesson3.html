Lesson Overview

Last time, you learned how to structure a web page with HTML. Now, let's explore more advanced HTML elements like forms, tables, and embedding media to make your pages interactive and informative.

In this lesson, we’ll explore more advanced HTML elements like forms, tables, and embedding media.

Forms

Forms allow users to submit data. They are created using the <form> tag. Here’s an example:

<form action="/submit" method="POST">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">

    <label for="email">Email:</label>
    <input type="email" id="email" name="email">

    <button type="submit">Submit</button>
</form>

Tables

Tables allow you to display data in rows and columns. Here’s an example:

<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>John</td>
        <td>25</td>
    </tr>
</table>

Embedding Media

HTML lets you embed images, videos, and other media using the <img> and <video> tags.

<img src="image.jpg" alt="A sample image">
