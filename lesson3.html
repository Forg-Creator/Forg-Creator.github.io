Lesson Overview

In this lesson, we will explore advanced HTML and CSS techniques, including linking pages, adding images, creating a navigation bar, and styling your website with CSS.

Linking Pages

To create a multi-page website, you can use the <a> tag to link different pages together. For example:

<a href="about.html">About Us</a>

Create an about.html page and link it from your main page using the code above.

Adding Images

Use the <img> tag to add images to your web pages. The src attribute specifies the image file, and the alt attribute provides alternative text:

<img src="example.jpg" alt="Example Image" width="300">

Make sure to include the image file in your project folder.

Creating a Navigation Bar

A navigation bar helps users move between pages on your website. Here's an example of a simple navigation bar:

<nav>
    <ul>
        <li><a href="index.html">Home</a></li>
        <li><a href="about.html">About</a></li>
        <li><a href="contact.html">Contact</a></li>
    </ul>
</nav>

Styling with CSS

CSS allows you to style your HTML elements. For example, you can change the background color, font size, and layout of your website. Here's an example:

body {
    font-family: Arial, sans-serif;
    background-color: #f4f4f4;
    color: #333;
}

h1 {
    color: #4CAF50;
    text-align: center;
}

Add the above CSS rules to your styles.css file to style your website.

Challenge

Create a small website with the following features:

  1. A homepage with links to "About" and "Contact" pages.
  2. An "About" page with a brief description and an image.
  3. A "Contact" page with a form for users to submit their information.
  4. Style your website using CSS to make it visually appealing.
