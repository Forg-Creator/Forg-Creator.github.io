Lesson 3: Introduction to CSS

CSS (Cascading Style Sheets) is used to style HTML elements and make them visually appealing.

CSS Syntax

CSS is made up of selectors, properties, and values. Here's an example:

    <style>
        h1 {
            color: blue;
        }
        p {
            font-size: 16px;
        }
    </style>

Practice:

Create a webpage that uses CSS to style your HTML tags. Here's an example:

    <html>
        <head>
            <style>
                h1 {
                    color: red;
                }
                p {
                    font-size: 18px;
                }
            </style>
        </head>
        <body>
            <h1>Styled Page</h1>
            <p>This page uses CSS to style text.</p>
        </body>
    </html>
