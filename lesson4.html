Lesson Overview

Previously, you learned about advanced HTML elements. Now, you'll discover how to use CSS to style your HTML pages and make them visually appealing.

This lesson introduces you to CSS, which is used to style your HTML content. You will learn how to change the look and feel of your page using various CSS properties.

Basic CSS Syntax

CSS uses selectors to target HTML elements and apply styles. Here’s an example:

body {
    background-color: lightgray;
    font-family: Arial, sans-serif;
}

h1 {
    color: blue;
    text-align: center;
}

Common CSS Properties

Here are some common CSS properties:

  • color - Sets the text color.
  • font-family - Sets the font.
  • background-color - Sets the background color of an element.
  • padding - Adds space inside an element.

Challenge

Apply the following styles to your page:

  2. Style the headings with a different color.
