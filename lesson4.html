Lesson 4: CSS Styling Basics

In this lesson, we will dive deeper into CSS styling techniques and how to apply them to different elements.

Styling Text

  • Font Family: font-family: Arial, sans-serif;
  • Font Size: font-size: 16px;
  • Text Color: color: #333;
  • Text Align: text-align: center;

CSS Box Model

The box model is a concept that explains how the dimensions of an element are calculated:

  • Content: The actual content inside the element
  • Padding: The space around the content
  • Border: The line surrounding the padding
  • Margin: The space outside the border

Practice:

Create a page with a box that contains text. Style it with borders, padding, and margin. Here's an example:

    <html>
        <head>
            <style>
                .box {
                    width: 300px;
                    padding: 20px;
                    border: 2px solid black;
                    margin: 30px;
                    text-align: center;
                }
            </style>
        </head>
        <body>
            <div class="box">
                <p>This is a box with padding, border, and margin.</p>
            </div>
        </body>
    </html>

Once you're done, move on to Lesson 5!
