Lesson Overview

In the last lesson, you learned the basics of CSS. Now, we’ll explore advanced CSS styling techniques, including typography, colors, and adding effects like borders and shadows.

Typography

Typography is essential for creating visually appealing text. Use the following CSS properties to style text:

h1 {
    font-family: 'Arial', sans-serif;
    font-size: 2.5em;
    font-weight: bold;
    color: #4CAF50;
    text-align: center;
}

Colors

Colors can make your website vibrant and engaging. Here's an example:

body {
    background-color: #f4f4f4;
    color: #333;
}

button {
    background-color: #4CAF50;
    color: white;
    border: none;
    padding: 10px 20px;
    border-radius: 5px;
    cursor: pointer;
}

button:hover {
    background-color: #45a049;
}
