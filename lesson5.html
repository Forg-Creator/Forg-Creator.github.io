Lesson Overview

In this lesson, we will explore advanced CSS layout techniques, including Flexbox and Grid, to create complex and responsive designs.

Flexbox

Flexbox is a powerful layout tool for aligning and distributing space among items in a container. Here's an example:

/* Container styles */
.container {
    display: flex;
    justify-content: space-between;
    align-items: center;
    padding: 10px;
    background-color: #f4f4f4;
}

/* Item styles */
.item {
    background-color: #4CAF50;
    color: white;
    padding: 20px;
    border-radius: 5px;
}

Use Flexbox to create layouts like navigation bars, card grids, and more.

CSS Grid

CSS Grid is a two-dimensional layout system for creating complex designs. Here's an example:

/* Container styles */
.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
    padding: 10px;
    background-color: #f4f4f4;
}

/* Item styles */
.grid-item {
    background-color: #4CAF50;
    color: white;
    padding: 20px;
    border-radius: 5px;
    text-align: center;
}

Use Grid to create layouts like photo galleries, dashboards, and more.

Combining Flexbox and Grid

You can combine Flexbox and Grid to create even more flexible designs. For example, use Grid for the overall layout and Flexbox for aligning items within a grid cell.

Challenge

Create a webpage with the following features:

  1. A navigation bar using Flexbox.
  2. A photo gallery using CSS Grid with at least 3 columns.
  3. Responsive adjustments using media queries to make the layout adapt to smaller screens.
