Lesson Overview

Last lesson, you learned how to add animations and transitions with CSS. Now, you'll make your website look great on any device by learning about responsive web design using media queries and flexible layouts.

Media Queries

Media queries let you apply CSS styles based on the device's screen size. For example:

@media (max-width: 600px) {
    body {
        font-size: 14px;
    }
}

Flexible Layouts

Use percentage widths, Flexbox, or Grid to make your layout adapt to different screen sizes.

Challenge

Update your website so it looks good on both desktop and mobile devices. Try resizing your browser to test your changes!
