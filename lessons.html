Lesson 1: Setting Up Visual Studio Code and GitHub

Learn how to install Visual Studio Code Insiders as your code editor and set up GitHub for version control.

Lesson 2: Introduction to HTML

Learn the basics of HTML structure, including creating elements like headings, paragraphs, and links.

Lesson 3: Introduction to CSS

Understand the basics of CSS, such as styling text, adding background colors, and positioning elements on the page.

Lesson 4: Responsive Web Design

Discover how to make your websites responsive, so they look great on devices of all sizes.

Lesson 5: Advanced HTML & CSS

Explore more advanced HTML elements and CSS features like Flexbox and Grid layouts.

Lesson 6: Advanced CSS & Animation

Focus on advanced CSS techniques like animations and transitions.

Lesson 7: Full Day of CSS

Enhance your skills in styling, layout design, and making responsive adjustments.

Lesson 8: Final Project

Pull together everything you have learned throughout the course to create a final website project. You will work on this project in bits and pieces throughout the entire course.
